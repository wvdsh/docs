The Leaderboards API lets you create score-based rankings for your game. Players can submit scores, view their rank, and compete with others.

## Getting a leaderboard

Retrieve an existing leaderboard by name:

<CodeGroup>
```gdscript Godot
func _ready():
    WavedashSDK.got_leaderboard.connect(_on_got_leaderboard)

func fetch_leaderboard():
    WavedashSDK.get_leaderboard("high-scores")

func _on_got_leaderboard(response):
    if response.get("success", false):
        var leaderboard = response["data"]
        print("Leaderboard: ", leaderboard["name"])
        print("Total entries: ", leaderboard["totalEntries"])
        print("Sort order: ", leaderboard["sortOrder"])
```

```csharp Unity
public class LeaderboardExample : MonoBehaviour
{
    async void Start()
    {
        var leaderboard = await Wavedash.SDK.GetLeaderboard("high-scores");
        if (leaderboard != null)
        {
            Debug.Log($"Leaderboard: {leaderboard["name"]}");
            Debug.Log($"Total entries: {leaderboard["totalEntries"]}");
            Debug.Log($"Sort order: {leaderboard["sortOrder"]}");
        }
    }
}
```

```javascript JavaScript
const response = await WavedashJS.getLeaderboard("high-scores");

if (response.success) {
  const leaderboard = response.data;
  console.log(`Leaderboard: ${leaderboard.name}`);
  console.log(`Total entries: ${leaderboard.totalEntries}`);
  console.log(`Sort order: ${leaderboard.sortOrder}`);
}
```
</CodeGroup>

## Creating a leaderboard

Create a leaderboard if it doesn't exist, or get it if it does:

<CodeGroup>
```gdscript Godot
func create_leaderboard():
    WavedashSDK.get_or_create_leaderboard(
        "speedrun-times",
        WavedashConstants.LEADERBOARD_SORT_ASCENDING,
        WavedashConstants.LEADERBOARD_DISPLAY_TYPE_TIME_MILLISECONDS
    )

func _on_got_leaderboard(response):
    if response.get("success", false):
        var leaderboard_id = response["data"]["id"]
        print("Leaderboard ID: ", leaderboard_id)
```

```csharp Unity
public async void CreateLeaderboard()
{
    var leaderboard = await Wavedash.SDK.GetOrCreateLeaderboard(
        "speedrun-times",
        sortMethod: WavedashConstants.LeaderboardSortMethod.ASCENDING,
        displayType: WavedashConstants.LeaderboardDisplayType.TIME_MILLISECONDS
    );

    if (leaderboard != null)
    {
        Debug.Log($"Leaderboard ID: {leaderboard["id"]}");
    }
}
```

```javascript JavaScript
const response = await WavedashJS.getOrCreateLeaderboard(
  "speedrun-times",  // Leaderboard name
  0,                 // Sort order: 0 = ascending, 1 = descending
  2                  // Display type: 0 = numeric, 1 = time_seconds, 2 = time_milliseconds, 3 = time_game_ticks
);

if (response.success) {
  const leaderboard = response.data;
  console.log(`Leaderboard ID: ${leaderboard.id}`);
}
```
</CodeGroup>

### Sort order

| Value | Constant | Description | Best for |
|-------|----------|-------------|----------|
| `0` | `LEADERBOARD_SORT_ORDER.ASC` | Lower scores rank higher | Time trials, golf scores |
| `1` | `LEADERBOARD_SORT_ORDER.DESC` | Higher scores rank higher | Points, high scores |

### Display type

| Value | Constant | Description |
|-------|----------|-------------|
| `0` | `LEADERBOARD_DISPLAY_TYPE.NUMERIC` | Display as a number (e.g., "1,000") |
| `1` | `LEADERBOARD_DISPLAY_TYPE.TIME_SECONDS` | Display as time in seconds (e.g., "1:23") |
| `2` | `LEADERBOARD_DISPLAY_TYPE.TIME_MILLISECONDS` | Display as time with milliseconds (e.g., "1:23.456") |
| `3` | `LEADERBOARD_DISPLAY_TYPE.TIME_GAME_TICKS` | Display as game ticks (assumes 60fps) |

## Submitting scores

Upload a player's score to a leaderboard:

<CodeGroup>
```gdscript Godot
func _ready():
    WavedashSDK.posted_leaderboard_score.connect(_on_score_posted)

func submit_score(leaderboard_id: String, score: int):
    # keep_best = true to only update if better than existing
    WavedashSDK.post_leaderboard_score(leaderboard_id, score, true)

func _on_score_posted(response):
    if response.get("success", false):
        var entry = response["data"]
        print("Rank: ", entry["rank"])
        print("Score: ", entry["score"])
```

```csharp Unity
public async void SubmitScore(string leaderboardId, int score)
{
    var result = await Wavedash.SDK.UploadLeaderboardScore(
        leaderboardId,
        score,
        keepBest: true  // Only update if better than existing
    );

    if (result != null)
    {
        Debug.Log($"Rank: {result["rank"]}");
        Debug.Log($"Score: {result["score"]}");
    }
}
```

```javascript JavaScript
const response = await WavedashJS.uploadLeaderboardScore(
  "j57abc123...",  // Leaderboard ID
  1500,            // Score value
  true             // keepBest: Only update if better than existing
);

if (response.success) {
  const entry = response.data;
  console.log(`Rank: ${entry.rank}`);
  console.log(`Score: ${entry.score}`);
}
```
</CodeGroup>

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `leaderboardId` | string | Yes | The leaderboard's ID |
| `score` | number | Yes | The score to submit |
| `keepBest` | boolean | Yes | If true, only updates if score is better |
| `ugcId` | string | No | Optional UGC attachment (replay, screenshot) |

<Tip>
Use `keepBest: true` to automatically handle personal best tracking. The SDK compares based on the leaderboard's sort order.
</Tip>

## Querying entries

### List top entries

Get the top entries from a leaderboard:

<CodeGroup>
```gdscript Godot
func _ready():
    WavedashSDK.got_leaderboard_entries.connect(_on_got_entries)

func get_top_scores(leaderboard_id: String):
    # offset, limit, friends_only
    WavedashSDK.get_leaderboard_entries(leaderboard_id, 0, 10, false)

func get_friends_scores(leaderboard_id: String):
    # Get only friends' scores
    WavedashSDK.get_leaderboard_entries(leaderboard_id, 0, 10, true)

func _on_got_entries(response):
    if response.get("success", false):
        for entry in response["data"]:
            print("#", entry["rank"], " ", entry["username"], ": ", entry["score"])
```

```csharp Unity
public async void GetTopScores(string leaderboardId)
{
    // Note: Unity SDK does not support friendsOnly filter
    var entries = await Wavedash.SDK.ListLeaderboardEntries(
        leaderboardId,
        offset: 0,
        limit: 10
    );

    foreach (var entry in entries)
    {
        Debug.Log($"#{entry["rank"]} {entry["username"]}: {entry["score"]}");
    }
}
```

```javascript JavaScript
const response = await WavedashJS.listLeaderboardEntries(
  "j57abc123...",  // Leaderboard ID
  0,               // offset: Start from rank 1
  10,              // limit: Get top 10
  false            // friendsOnly: false for all players, true for friends only
);

if (response.success) {
  response.data.forEach(entry => {
    console.log(`#${entry.rank} ${entry.username}: ${entry.score}`);
  });
}

// Get only friends' scores
const friendsResponse = await WavedashJS.listLeaderboardEntries(
  "j57abc123...",
  0,
  10,
  true  // friendsOnly
);
```
</CodeGroup>

### List entries around user

Get entries around the current player's rank:

<CodeGroup>
```gdscript Godot
func get_scores_around_me(leaderboard_id: String):
    # count_ahead, count_behind, friends_only
    WavedashSDK.get_leaderboard_entries_around_player(leaderboard_id, 5, 5, false)

func get_friend_scores_around_me(leaderboard_id: String):
    # Get friends' scores around your position
    WavedashSDK.get_leaderboard_entries_around_player(leaderboard_id, 5, 5, true)

func _on_got_entries(response):
    if response.get("success", false):
        var my_id = WavedashSDK.get_user_id()
        for entry in response["data"]:
            var is_me = entry["userId"] == my_id
            var prefix = "> " if is_me else "  "
            print(prefix, "#", entry["rank"], " ", entry["username"], ": ", entry["score"])
```

```csharp Unity
public async void GetScoresAroundMe(string leaderboardId)
{
    // Note: Unity SDK does not support friendsOnly filter
    var entries = await Wavedash.SDK.ListLeaderboardEntriesAroundUser(
        leaderboardId,
        countAhead: 5,
        countBehind: 5
    );

    var myId = Wavedash.SDK.GetUserId();
    foreach (var entry in entries)
    {
        var isMe = entry["userId"].ToString() == myId;
        var prefix = isMe ? "> " : "  ";
        Debug.Log($"{prefix}#{entry["rank"]} {entry["username"]}: {entry["score"]}");
    }
}
```

```javascript JavaScript
const response = await WavedashJS.listLeaderboardEntriesAroundUser(
  "j57abc123...",  // Leaderboard ID
  5,               // countAhead: 5 entries with better scores
  5,               // countBehind: 5 entries with worse scores
  false            // friendsOnly: false for all players, true for friends only
);

if (response.success) {
  // Returns up to 11 entries centered on the player
  response.data.forEach(entry => {
    const isMe = entry.userId === WavedashJS.getUserId();
    console.log(`${isMe ? ">" : " "} #${entry.rank} ${entry.username}: ${entry.score}`);
  });
}

// Get friends' scores around your position
const friendsResponse = await WavedashJS.listLeaderboardEntriesAroundUser(
  "j57abc123...",
  5,
  5,
  true  // friendsOnly
);
```
</CodeGroup>

### Get player's entry

Get the current player's leaderboard entry:

<CodeGroup>
```gdscript Godot
func get_my_entry(leaderboard_id: String):
    WavedashSDK.get_my_leaderboard_entries(leaderboard_id)

func _on_got_my_entries(response):
    if response.get("success", false) and response["data"].size() > 0:
        var my_entry = response["data"][0]
        print("Your rank: #", my_entry["rank"])
        print("Your score: ", my_entry["score"])
```

```csharp Unity
public async void GetMyEntry(string leaderboardId)
{
    var entries = await Wavedash.SDK.GetMyLeaderboardEntries(leaderboardId);

    if (entries != null && entries.Count > 0)
    {
        var myEntry = entries[0];
        Debug.Log($"Your rank: #{myEntry["rank"]}");
        Debug.Log($"Your score: {myEntry["score"]}");
    }
}
```

```javascript JavaScript
const response = await WavedashJS.getMyLeaderboardEntries("j57abc123...");

if (response.success && response.data.length > 0) {
  const myEntry = response.data[0];
  console.log(`Your rank: #${myEntry.rank}`);
  console.log(`Your score: ${myEntry.score}`);
}
```
</CodeGroup>

## Entry count

Get the total number of entries (cached, synchronous):

<CodeGroup>
```gdscript Godot
var count = WavedashSDK.get_leaderboard_entry_count(leaderboard_id)
print("Total players: ", count)
```

```csharp Unity
var count = Wavedash.SDK.GetLeaderboardEntryCount(leaderboardId);
Debug.Log($"Total players: {count}");
```

```javascript JavaScript
const count = WavedashJS.getLeaderboardEntryCount("j57abc123...");
console.log(`Total players: ${count}`);
```
</CodeGroup>

<Note>
This returns a cached value from the last query. Returns -1 if the leaderboard hasn't been queried yet.
</Note>

## Leaderboard entry structure

```typescript
interface LeaderboardEntry {
  userId: string;      // Player's user ID
  username: string;    // Player's display name
  score: number;       // Score value
  rank: number;        // Current rank
  ugcId?: string;      // Attached UGC (if any)
  createdAt: number;   // When first submitted
  updatedAt: number;   // When last updated
}
```

## Attaching replays or screenshots

You can attach user-generated content (like replays or screenshots) to leaderboard entries:

<CodeGroup>
```gdscript Godot
func submit_score_with_replay(leaderboard_id: String, score: int, ugc_id: String):
    # Submit score with UGC attachment
    WavedashSDK.post_leaderboard_score(leaderboard_id, score, true, ugc_id)
```

```csharp Unity
public async void SubmitScoreWithReplay(string leaderboardId, int score, string ugcId)
{
    var result = await Wavedash.SDK.UploadLeaderboardScore(
        leaderboardId,
        score,
        keepBest: true,
        ugcId: ugcId
    );
}
```

```javascript JavaScript
// First, create the UGC item with file path
const ugcResponse = await WavedashJS.createUGCItem(
  3,                     // ugcType: 3 = game_managed (replays)
  "My Replay",           // title
  "",                    // description
  0,                     // visibility: 0 = public
  "/path/to/replay.dat"  // filePath (optional)
);

if (ugcResponse.success) {
  // Then submit score with UGC attachment
  const scoreResponse = await WavedashJS.uploadLeaderboardScore(
    "speedrun-times",     // leaderboardId
    12345,                // score
    true,                 // keepBest
    ugcResponse.data      // ugcId
  );
}
```
</CodeGroup>

See [User-Generated Content](/sdk/ugc) for more on UGC.

## Best practices

<AccordionGroup>
  <Accordion title="Use descriptive leaderboard names">
    Use clear, unique names like "level-1-speedrun" or "weekly-high-score" rather than generic names.
  </Accordion>
  <Accordion title="Always use keepBest for competitive leaderboards">
    Set `keepBest: true` to automatically track personal bests. This prevents overwriting a better score with a worse one.
  </Accordion>
  <Accordion title="Cache leaderboard IDs">
    Store the leaderboard ID after calling `getOrCreateLeaderboard` to avoid extra network calls.
  </Accordion>
  <Accordion title="Handle offline gracefully">
    Check `response.success` and show appropriate UI when submissions fail.
  </Accordion>
</AccordionGroup>
