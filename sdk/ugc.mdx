title: "User-Generated Content"
description: "Enable players to upload and share custom content"

The UGC (User-Generated Content) API lets players upload and share content like replays, screenshots, custom levels, or other game-specific files.

## Creating a UGC item

Create a new UGC item to get an upload URL:

<CodeGroup>
```gdscript Godot
func _ready():
    WavedashSDK.ugc_item_created.connect(_on_ugc_created)

func create_screenshot_ugc(local_path: String):
    WavedashSDK.create_ugc_item(
        WavedashConstants.UGC_TYPE_SCREENSHOT,
        "My Screenshot",
        "A cool moment in the game",
        WavedashConstants.UGC_VISIBILITY_PUBLIC,
        local_path
    )

func _on_ugc_created(response):
    if response.get("success", false):
        var ugc_id = response["data"]["id"]
        print("UGC created: ", ugc_id)
```

```csharp Unity
using Wavedash;

public class UGCExample : MonoBehaviour
{
    public async void CreateScreenshotUGC(string localPath)
    {
        var result = await SDK.CreateUGCItem(
            ugcType: 0,        // 0 = screenshot
            title: "My Screenshot",
            description: "A cool moment in the game",
            visibility: 0,     // 0 = public
            localPath: localPath
        );

        if (result != null)
        {
            Debug.Log($"UGC created: {result["id"]}");
        }
    }
}
```

```javascript JavaScript
const response = await WavedashJS.createUGCItem({
  ugcType: "replay",
  visibility: "public",
  metadata: {
    levelName: "World 1-1",
    duration: 125
  }
});

if (response.success) {
  const ugcItem = response.data;
  console.log(`UGC ID: ${ugcItem.id}`);
  console.log(`Upload URL: ${ugcItem.uploadUrl}`);
}
```
</CodeGroup>

### Visibility options

| Value | Description |
|-------|-------------|
| `public` | Anyone can view |
| `private` | Only the creator can view |
| `friends` | Only friends can view |

## Downloading content

<CodeGroup>
```gdscript Godot
func _ready():
    WavedashSDK.ugc_item_downloaded.connect(_on_ugc_downloaded)

func download_ugc(ugc_id: String, local_path: String):
    WavedashSDK.download_ugc_item(ugc_id, local_path)

func _on_ugc_downloaded(response):
    if response.get("success", false):
        print("UGC downloaded to local path")
```

```csharp Unity
public async void DownloadUGC(string ugcId, string localPath)
{
    var result = await SDK.DownloadUGCItem(ugcId, localPath);

    if (result != null)
    {
        Debug.Log("UGC downloaded to local path");
    }
}
```

```javascript JavaScript
const response = await WavedashJS.downloadUGCItem("ugc_abc123...");

if (response.success) {
  const data = response.data; // Uint8Array
}
```
</CodeGroup>

## UGC types

| Type | Description |
|------|-------------|
| `replay` | Game replay data |
| `screenshot` | Image capture |
| `level` | Custom level or map |
| `save` | Save game data |
| `skin` | Custom character/item skin |

## Example: Screenshot sharing

<CodeGroup>
```gdscript Godot
func share_screenshot():
    var image = get_viewport().get_texture().get_image()
    var path = "user://screenshots/capture.png"

    DirAccess.make_dir_recursive_absolute("user://screenshots")
    image.save_png(path)

    WavedashSDK.create_ugc_item(
        WavedashConstants.UGC_TYPE_SCREENSHOT,
        "Game Screenshot",
        "Captured at level " + str(current_level),
        WavedashConstants.UGC_VISIBILITY_PUBLIC,
        path
    )
```

```csharp Unity
public async void ShareScreenshot()
{
    string path = $"{Application.persistentDataPath}/screenshots/capture.png";
    Directory.CreateDirectory(Path.GetDirectoryName(path));
    ScreenCapture.CaptureScreenshot(path);

    await Task.Yield();

    var result = await SDK.CreateUGCItem(
        ugcType: 0,
        title: "Game Screenshot",
        description: $"Captured at level {currentLevel}",
        visibility: 0,
        localPath: path
    );
}
```

```javascript JavaScript
async function shareScreenshot(imageData) {
  const response = await WavedashJS.createUGCItem({
    ugcType: "screenshot",
    visibility: "public",
    metadata: { width: 1920, height: 1080, level: currentLevel }
  });

  if (response.success) {
    await fetch(response.data.uploadUrl, {
      method: "PUT",
      body: imageData,
      headers: { "Content-Type": "image/png" }
    });
    return response.data.id;
  }
  return null;
}
```
