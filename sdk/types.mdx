This page documents the TypeScript types used throughout the Wavedash SDK.

## Response types

### WavedashResponse

All SDK methods return a `WavedashResponse` wrapper:

```typescript
interface WavedashResponse<T> {
  success: boolean;     // Whether the operation succeeded
  data: T | null;       // Result data (null on failure)
  args: Record<string, any>;  // Original arguments passed
  message?: string;     // Error message if success is false
}
```

**Example usage:**

```typescript
const response = await WavedashJS.getLeaderboard("high-scores");

if (response.success) {
  const leaderboard: Leaderboard = response.data;
  // Use leaderboard...
} else {
  console.error(response.message);
}
```

## Configuration types

### WavedashConfig

SDK configuration options:

```typescript
interface WavedashConfig {
  gameId: string;              // Your game's ID
  debug?: boolean;             // Enable debug logging
  remoteStorageOrigin?: string; // Custom storage origin
  p2p?: Partial<P2PConfig>;    // P2P options
  disableAchievementsAndStats?: boolean; // Disable stats
  deferEvents?: boolean;       // Defer events until ready
}
```

### P2PConfig

Peer-to-peer networking configuration:

```typescript
interface P2PConfig {
  maxPeers: number;              // Max concurrent connections
  enableReliableChannel: boolean;  // TCP-like channel
  enableUnreliableChannel: boolean; // UDP-like channel
}
```

## Leaderboard types

### Leaderboard

```typescript
interface Leaderboard {
  id: string;           // Leaderboard ID
  name: string;         // Display name
  sortOrder: LeaderboardSortOrder;   // "ascending" | "descending"
  displayType: LeaderboardDisplayType; // "numeric" | "time"
  totalEntries: number; // Total number of entries
}
```

### LeaderboardSortOrder

```typescript
type LeaderboardSortOrder = "ascending" | "descending";
```

### LeaderboardDisplayType

```typescript
type LeaderboardDisplayType = "numeric" | "time";
```

### LeaderboardEntry

```typescript
interface LeaderboardEntry {
  userId: string;      // Player's user ID
  username: string;    // Player's display name
  score: number;       // Score value
  rank: number;        // Current rank
  ugcId?: string;      // Attached UGC (optional)
  createdAt: number;   // First submission timestamp
  updatedAt: number;   // Last update timestamp
}
```

## Engine types

### EngineInstance

Interface for game engine instances:

```typescript
interface EngineInstance {
  type: "godot" | "unity" | "custom";

  // Send message to engine
  SendMessage(
    objectName: string,
    methodName: string,
    value?: string | number | boolean
  ): void;

  // Emscripten filesystem API
  FS: {
    readFile(path: string, opts?: Record<string, any>): string | Uint8Array;
    writeFile(path: string, data: string | ArrayBufferView, opts?: Record<string, any>): void;
    mkdirTree(path: string, mode?: number): void;
    syncfs(populate: boolean, callback?: (err: any) => void): void;
    analyzePath(path: string): { exists: boolean };
  };
}
```
