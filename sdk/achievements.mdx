The Achievements API lets you track player statistics and award achievements based on their progress. Stats persist across sessions and sync automatically.

<Note>
Achievements are currently created by the Wavedash team. Contact us at [developer@wavedash.com](mailto:developer@wavedash.com) or via [Discord](https://discord.gg/bBaWUjTJWx) to set up achievements for your game. Dashboard creation is coming soon.
</Note>

## Stats

Stats are numeric values that track player progress, like total kills, distance traveled, or games played.

### Setting a stat

<CodeGroup>
```gdscript Godot
# Set a stat value
# store_now = true will immediately sync to server
WavedashSDK.set_stat_int("total_kills", 150, true)
WavedashSDK.set_stat_int("distance_traveled", 5280, true)
WavedashSDK.set_stat_int("games_played", 42, true)
```

```csharp Unity
// Set stat values
SDK.SetStat("total_kills", 150);
SDK.SetStat("distance_traveled", 5280.5f);
SDK.SetStat("games_played", 42);
```

```javascript JavaScript
WavedashJS.setStat("total_kills", 150);
WavedashJS.setStat("distance_traveled", 5280.5);
WavedashJS.setStat("games_played", 42);
```
</CodeGroup>

<Note>
Stat updates are debounced to reduce network calls. Changes sync automatically.
</Note>

### Getting a stat

<CodeGroup>
```gdscript Godot
var kills = WavedashSDK.get_stat_int("total_kills")
print("Total kills: ", kills)
```

```csharp Unity
var kills = SDK.GetStat("total_kills");
Debug.Log($"Total kills: {kills}");
```

```javascript JavaScript
const kills = WavedashJS.getStat("total_kills");
console.log(`Total kills: ${kills}`);
```
</CodeGroup>

### Incrementing a stat

<CodeGroup>
```gdscript Godot
func add_coins(amount: int):
    var current = WavedashSDK.get_stat_int("coins_collected")
    WavedashSDK.set_stat_int("coins_collected", current + amount, true)
```

```csharp Unity
public void AddCoins(int amount)
{
    var current = SDK.GetStat("coins_collected");
    SDK.SetStat("coins_collected", current + amount);
}
```

```javascript JavaScript
const current = WavedashJS.getStat("coins_collected") || 0;
WavedashJS.setStat("coins_collected", current + 10);
```
</CodeGroup>

## Achievements

Achievements are unlockable awards that recognize player accomplishments.

### Unlocking an achievement

<CodeGroup>
```gdscript Godot
func unlock_first_win():
    WavedashSDK.set_achievement("first_win")
    print("Achievement unlocked!")
```

```csharp Unity
public void UnlockFirstWin()
{
    SDK.SetAchievement("first_win");
    Debug.Log("Achievement unlocked!");
}
```

```javascript JavaScript
const response = await WavedashJS.setAchievement("first_win");

if (response.success) {
  console.log("Achievement unlocked!");
}
```
</CodeGroup>

### Checking achievement status

<CodeGroup>
```gdscript Godot
func check_achievement(achievement_name: String) -> bool:
    var is_unlocked = WavedashSDK.get_achievement(achievement_name)
    if is_unlocked:
        print("Player has already earned this achievement")
    return is_unlocked
```

```csharp Unity
public bool CheckAchievement(string achievementName)
{
    var isUnlocked = SDK.GetAchievement(achievementName);
    if (isUnlocked)
    {
        Debug.Log("Player has already earned this achievement");
    }
    return isUnlocked;
}
```

```javascript JavaScript
const isUnlocked = WavedashJS.getAchievement("first_win");

if (isUnlocked) {
  console.log("Player has already earned this achievement");
}
```
</CodeGroup>

### Achievements with progress

Some achievements are tied to stats. When a stat reaches a threshold, the achievement unlocks automatically.

<CodeGroup>
```gdscript Godot
# Update the stat - achievement unlocks automatically if threshold is met
func on_enemy_defeated():
    var enemies = WavedashSDK.get_stat_int("enemies_defeated") + 1
    WavedashSDK.set_stat_int("enemies_defeated", enemies, true)
```

```csharp Unity
public void OnEnemyDefeated()
{
    var enemies = SDK.GetStat("enemies_defeated") + 1;
    SDK.SetStat("enemies_defeated", enemies);
}
```

```javascript JavaScript
WavedashJS.setStat("enemies_defeated", 100);
// Achievement unlocks automatically if threshold is met
```
</CodeGroup>

## Stat-linked achievements

Configure achievements in the dashboard that automatically unlock when a stat reaches a value:

| Achievement | Stat | Threshold |
|-------------|------|-----------|
| "First Blood" | kills | 1 |
| "Centurion" | kills | 100 |
| "Marathon" | distance_traveled | 26219 |

## Example: Kill tracking

<CodeGroup>
```gdscript Godot
var session_kills: int = 0

func on_enemy_killed():
    var kills = WavedashSDK.get_stat_int("total_kills") + 1
    WavedashSDK.set_stat_int("total_kills", kills, true)
    session_kills += 1

    if kills == 1:
        WavedashSDK.set_achievement("first_blood")
```

```csharp Unity
private int sessionKills = 0;

public void OnEnemyKilled()
{
    var kills = SDK.GetStat("total_kills") + 1;
    SDK.SetStat("total_kills", kills);
    sessionKills++;

    if (kills == 1)
    {
        SDK.SetAchievement("first_blood");
    }
}
```

```javascript JavaScript
function onEnemyKilled() {
  const kills = (WavedashJS.getStat("total_kills") || 0) + 1;
  WavedashJS.setStat("total_kills", kills);
  sessionKills++;

  if (kills === 1) {
    WavedashJS.setAchievement("first_blood");
  }
}
```
</CodeGroup>

## Example: Score tracking

<CodeGroup>
```gdscript Godot
func on_game_over(score: int):
    var high_score = WavedashSDK.get_stat_int("high_score")

    if score > high_score:
        WavedashSDK.set_stat_int("high_score", score, true)
        print("New high score!")

    var games_played = WavedashSDK.get_stat_int("games_played") + 1
    WavedashSDK.set_stat_int("games_played", games_played, true)
```

```csharp Unity
public void OnGameOver(int score)
{
    var highScore = SDK.GetStat("high_score");

    if (score > highScore)
    {
        SDK.SetStat("high_score", score);
        Debug.Log("New high score!");
    }

    var gamesPlayed = SDK.GetStat("games_played") + 1;
    SDK.SetStat("games_played", gamesPlayed);
}
```

```javascript JavaScript
function onGameOver(score) {
  const highScore = WavedashJS.getStat("high_score") || 0;

  if (score > highScore) {
    WavedashJS.setStat("high_score", score);
    console.log("New high score!");
  }

  const gamesPlayed = (WavedashJS.getStat("games_played") || 0) + 1;
  WavedashJS.setStat("games_played", gamesPlayed);
}
```
</CodeGroup>
