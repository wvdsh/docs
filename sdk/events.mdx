---
title: "Events Reference"
description: "Events emitted by the Wavedash SDK to your game"
---

Events are notifications that the SDK emits to your game in real time. Your game engine receives these events to react to lobby changes, P2P connections, and backend status.

## Event overview

| Event | Category | Description |
|-------|----------|-------------|
| `WavedashJS.Events.LOBBY_JOINED` | Lobby | Successfully joined a lobby |
| `WavedashJS.Events.LOBBY_USERS_UPDATED` | Lobby | Users joined or left the lobby |
| `WavedashJS.Events.LOBBY_MESSAGE` | Lobby | New message received |
| `WavedashJS.Events.LOBBY_DATA_UPDATED` | Lobby | Lobby metadata changed |
| `WavedashJS.Events.LOBBY_KICKED` | Lobby | Removed from lobby |
| `WavedashJS.Events.LOBBY_INVITE` | Lobby | Received lobby invitation |
| `WavedashJS.Events.P2P_CONNECTION_ESTABLISHED` | P2P | Connected to a peer |
| `WavedashJS.Events.P2P_CONNECTION_FAILED` | P2P | Connection to peer failed |
| `WavedashJS.Events.P2P_PEER_DISCONNECTED` | P2P | Peer disconnected |
| `WavedashJS.Events.BACKEND_CONNECTED` | Backend | Connected to Wavedash |
| `WavedashJS.Events.BACKEND_DISCONNECTED` | Backend | Disconnected from Wavedash |
| `WavedashJS.Events.BACKEND_RECONNECTING` | Backend | Attempting to reconnect |

## Lobby events

### `WavedashJS.Events.LOBBY_JOINED`

Emitted when you successfully join or create a lobby.

```typescript
{
  success: boolean;
  lobbyId: string;           // Lobby ID
  hostId?: string;           // Host's user ID (present on success)
  users?: LobbyUser[];       // Current lobby members (present on success)
  metadata?: Record<string, unknown>; // Lobby metadata (present on success)
  message?: string;          // Error message (present on failure)
}
```

### `WavedashJS.Events.LOBBY_USERS_UPDATED`

Emitted when a user joins or leaves the lobby. Fires once per user change.

```typescript
{
  userId: string;                    // User ID
  username: string;                 // Display name
  changeType: "JOINED" | "LEFT";   // Whether the user joined or left
}
```

### `WavedashJS.Events.LOBBY_MESSAGE`

Emitted when a new message is received in the lobby.

```typescript
{
  messageId: string;
  userId: string;
  username: string;
  content: string;
  createdAt: number;
}
```

### `WavedashJS.Events.LOBBY_DATA_UPDATED`

Emitted when the lobby metadata changes. Contains the full metadata object.

```typescript
Record<string, unknown> // The full lobby metadata object
```

### `WavedashJS.Events.LOBBY_KICKED`

Emitted when you are removed from the lobby.

```typescript
{
  lobbyId: string;
  reason: "KICKED" | "ERROR";
}
```

### `WavedashJS.Events.LOBBY_INVITE`

Emitted when you receive an invitation to join a lobby.

```typescript
{
  lobbyId: string;
  inviterId: string;
  inviterUsername: string;
}
```

## P2P events

### `WavedashJS.Events.P2P_CONNECTION_ESTABLISHED`

Emitted when a peer-to-peer connection is ready.

```typescript
{
  userId: string;
  username: string;
}
```

### `WavedashJS.Events.P2P_CONNECTION_FAILED`

Emitted when a peer connection fails.

```typescript
{
  userId: string;
  username: string;
  error: string;
}
```

### `WavedashJS.Events.P2P_PEER_DISCONNECTED`

Emitted when a peer disconnects.

```typescript
{
  userId: string;
  username: string;
}
```

## Backend events

### `WavedashJS.Events.BACKEND_CONNECTED`

Emitted when connected to the Wavedash backend.

```typescript
{
  isConnected: boolean;
  hasEverConnected: boolean;
  connectionCount: number;
  connectionRetries: number;
}
```

### `WavedashJS.Events.BACKEND_DISCONNECTED`

Emitted when disconnected from the backend.

```typescript
{
  isConnected: boolean;
  hasEverConnected: boolean;
  connectionCount: number;
  connectionRetries: number;
}
```

### `WavedashJS.Events.BACKEND_RECONNECTING`

Emitted when attempting to reconnect after a disconnect.

```typescript
{
  isConnected: boolean;
  hasEverConnected: boolean;
  connectionCount: number;
  connectionRetries: number;
}
```

## Handling events

<Tabs>
  <Tab title="Godot">
    The Wavedash Godot SDK exposes events natively:

    ```gdscript
    extends Node

    func _ready():
        # Connect to lobby events
        WavedashSDK.lobby_joined.connect(_on_lobby_joined)
        WavedashSDK.lobby_message.connect(_on_lobby_message)
        WavedashSDK.lobby_users_updated.connect(_on_users_updated)
        WavedashSDK.lobby_kicked.connect(_on_lobby_kicked)

        # Connect to P2P events
        WavedashSDK.p2p_connection_established.connect(_on_peer_connected)
        WavedashSDK.p2p_peer_disconnected.connect(_on_peer_disconnected)

        # Connect to backend events
        WavedashSDK.backend_connected.connect(_on_backend_connected)
        WavedashSDK.backend_disconnected.connect(_on_backend_disconnected)

    func _on_lobby_joined(data: Dictionary):
        if data.success:
            print("Joined lobby: ", data.data)

    func _on_lobby_message(message: Dictionary):
        print(message.username, ": ", message.content)

    func _on_users_updated(data: Dictionary):
        for user in data.users:
            print("User in lobby: ", user.username)

    func _on_lobby_kicked(data: Dictionary):
        print("Kicked from lobby")
        get_tree().change_scene_to_file("res://menu.tscn")

    func _on_peer_connected(data: Dictionary):
        print("Connected to peer: ", data.username)

    func _on_peer_disconnected(data: Dictionary):
        print("Peer disconnected: ", data.userId)

    func _on_backend_connected(payload):
        print("Connected to Wavedash!")

    func _on_backend_disconnected(payload):
        print("Lost connection to Wavedash")
    ```

  <Tab title="Unity">
    The Unity SDK exposes events as C# events:

    ```csharp
    using UnityEngine;
    using System.Collections.Generic;

    public class WavedashEventHandler : MonoBehaviour
    {
        void Awake()
        {
            // Connect to lobby events
            Wavedash.SDK.OnLobbyJoined += OnLobbyJoined;
            Wavedash.SDK.OnLobbyMessage += OnLobbyMessage;
            Wavedash.SDK.OnLobbyUsersUpdated += OnUsersUpdated;
            Wavedash.SDK.OnLobbyKicked += OnLobbyKicked;

            // Connect to P2P events
            Wavedash.SDK.OnP2PConnectionEstablished += OnPeerConnected;
            Wavedash.SDK.OnP2PConnectionFailed += OnPeerFailed;
            Wavedash.SDK.OnP2PPeerDisconnected += OnPeerDisconnected;

            // Connect to backend events
            Wavedash.SDK.OnBackendConnected += OnBackendConnected;
            Wavedash.SDK.OnBackendDisconnected += OnBackendDisconnected;
            Wavedash.SDK.OnBackendReconnecting += OnBackendReconnecting;
        }

        void OnLobbyJoined(Dictionary<string, object> data)
        {
            Debug.Log($"Joined lobby: {data["lobbyId"]}");
        }

        void OnLobbyMessage(Dictionary<string, object> data)
        {
            Debug.Log($"{data["username"]}: {data["content"]}");
        }

        void OnUsersUpdated(Dictionary<string, object> data)
        {
            Debug.Log($"User update: {data["changeType"]}");
        }

        void OnLobbyKicked(Dictionary<string, object> data)
        {
            Debug.Log("Kicked from lobby");
        }

        void OnPeerConnected(Dictionary<string, object> data)
        {
            Debug.Log($"Connected to: {data["username"]}");
        }

        void OnPeerFailed(Dictionary<string, object> data)
        {
            Debug.Log($"Connection failed to: {data["userId"]}");
        }

        void OnPeerDisconnected(Dictionary<string, object> data)
        {
            Debug.Log($"Peer disconnected: {data["userId"]}");
        }

        void OnBackendConnected(Dictionary<string, object> data)
        {
            Debug.Log("Connected to Wavedash!");
        }

        void OnBackendDisconnected(Dictionary<string, object> data)
        {
            Debug.Log("Lost connection to Wavedash");
        }

        void OnBackendReconnecting(Dictionary<string, object> data)
        {
            Debug.Log("Reconnecting to Wavedash...");
        }

        void OnDestroy()
        {
            // Unsubscribe from events
            Wavedash.SDK.OnLobbyJoined -= OnLobbyJoined;
            Wavedash.SDK.OnLobbyMessage -= OnLobbyMessage;
            Wavedash.SDK.OnLobbyUsersUpdated -= OnUsersUpdated;
            Wavedash.SDK.OnLobbyKicked -= OnLobbyKicked;
            Wavedash.SDK.OnP2PConnectionEstablished -= OnPeerConnected;
            Wavedash.SDK.OnP2PConnectionFailed -= OnPeerFailed;
            Wavedash.SDK.OnP2PPeerDisconnected -= OnPeerDisconnected;
            Wavedash.SDK.OnBackendConnected -= OnBackendConnected;
            Wavedash.SDK.OnBackendDisconnected -= OnBackendDisconnected;
            Wavedash.SDK.OnBackendReconnecting -= OnBackendReconnecting;
        }
    }
    ```

  <Tab title="JavaScript">
    The SDK extends `EventTarget`, so you can use standard `addEventListener`:

    ```javascript
    // Listen for lobby events
    WavedashJS.addEventListener(WavedashJS.Events.LOBBY_JOINED, (e) => {
      const data = e.detail;
      if (data.success) {
        console.log("Joined lobby:", data.lobbyId);
      }
    });

    WavedashJS.addEventListener(WavedashJS.Events.LOBBY_MESSAGE, (e) => {
      const message = e.detail;
      console.log(`${message.username}: ${message.content}`);
    });

    WavedashJS.addEventListener(WavedashJS.Events.LOBBY_USERS_UPDATED, (e) => {
      const data = e.detail;
      console.log(`User ${data.changeType}: ${data.username}`);
    });

    WavedashJS.addEventListener(WavedashJS.Events.LOBBY_KICKED, (e) => {
      console.log("Kicked from lobby:", e.detail.reason);
    });

    // Listen for P2P events
    WavedashJS.addEventListener(WavedashJS.Events.P2P_CONNECTION_ESTABLISHED, (e) => {
      console.log("Connected to peer:", e.detail.username);
    });

    WavedashJS.addEventListener(WavedashJS.Events.P2P_PEER_DISCONNECTED, (e) => {
      console.log("Peer disconnected:", e.detail.userId);
    });

    // Listen for backend events
    WavedashJS.addEventListener(WavedashJS.Events.BACKEND_CONNECTED, (e) => {
      console.log("Connected to Wavedash!");
    });

    WavedashJS.addEventListener(WavedashJS.Events.BACKEND_DISCONNECTED, (e) => {
      console.log("Lost connection to Wavedash");
    });
    ```

## Deferring events

If your game needs time to load assets, set up scenes, or wire up event handlers before handling events, use `deferEvents: true` when calling `Init`. The SDK queues all events until your game calls `readyForEvents()`.

<CodeGroup>
```gdscript Godot
func _ready():
    # Initialize with deferEvents so events are queued during setup
    WavedashSDK.init({"debug": true, "deferEvents": true})

    # Wire up event handlers
    WavedashSDK.lobby_joined.connect(_on_lobby_joined)
    WavedashSDK.lobby_message.connect(_on_lobby_message)
    WavedashSDK.p2p_connection_established.connect(_on_p2p_connected)

    # Do any other game setup — load scenes, assets, etc.
    _load_game_resources()

    # Now tell the SDK you're ready to receive queued events
    WavedashSDK.ready_for_events()
```

```csharp Unity
void Awake()
{
    // Initialize with deferEvents so events are queued during setup
    Wavedash.SDK.Init(new Dictionary<string, object>
    {
        { "debug", true },
        { "deferEvents", true }
    });

    // Wire up event handlers
    Wavedash.SDK.OnLobbyJoined += HandleLobbyJoined;
    Wavedash.SDK.OnLobbyMessage += HandleLobbyMessage;
    Wavedash.SDK.OnP2PConnectionEstablished += HandleP2PConnected;
}

void Start()
{
    // Do any other game setup — load scenes, assets, etc.
    LoadGameResources();

    // Now tell the SDK you're ready to receive queued events
    Wavedash.SDK.ReadyForEvents();
}
```

```javascript JavaScript
// Initialize with deferEvents so events are queued during setup
WavedashJS.init({ debug: true, deferEvents: true });

// Set up your event listeners
WavedashJS.addEventListener(WavedashJS.Events.LOBBY_JOINED, (e) => {
  handleLobbyJoined(e.detail);
});
WavedashJS.addEventListener(WavedashJS.Events.LOBBY_MESSAGE, (e) => {
  handleLobbyMessage(e.detail);
});

// Do any other game setup — load assets, etc.
await loadGameResources();

// Now tell the SDK you're ready to receive queued events
WavedashJS.readyForEvents();
```
</CodeGroup>

This ensures no events are missed during initialization.
