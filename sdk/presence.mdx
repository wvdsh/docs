---
title: "Presence"
description: "Track and share player online status and activity"
---

The Presence API lets you share player status and activity with friends and other players. Use it to show what players are doing, enable "join game" functionality, and build social features.

## Updating presence

<CodeGroup>
```gdscript Godot
# Presence is not yet available in the Godot SDK.
# This feature is currently only available in the JavaScript SDK.
```

```csharp Unity
// UpdateUserPresence is not yet available in the Unity SDK.
// Presence features are currently only available in the JavaScript SDK.
```

```javascript JavaScript
await WavedashJS.updateUserPresence({
  status: "In Game",
  details: "Playing Level 5",
  level: 5,
  score: 12500,
  lobbyId: "lobby_abc123"
});
```
</CodeGroup>

### Parameters

The `updateUserPresence` function accepts an object with custom key-value pairs:

| Parameter | Type | Description |
|-----------|------|-------------|
| `status` | string | Short status text (e.g., "In Menu", "In Game") |
| `details` | string | Detailed activity description |
| `lobbyId` | string | Include to enable "Join Game" functionality |
| `...` | any | Any additional custom data |

## Use cases

### Menu vs in-game status

<CodeGroup>
```gdscript Godot
# Presence is not yet available in the Godot SDK.
```

```csharp Unity
// UpdateUserPresence is not yet available in the Unity SDK.
```

```javascript JavaScript
// In menu
await WavedashJS.updateUserPresence({ status: "In Menu", details: "Main Menu" });

// In game
await WavedashJS.updateUserPresence({
  status: "In Game",
  details: "Deathmatch on Arena",
  mode: "deathmatch",
  map: "arena"
});

// In lobby
await WavedashJS.updateUserPresence({
  status: "In Lobby",
  details: "Waiting (2/4)",
  lobbyId: currentLobbyId,
  canJoin: true
});
```
</CodeGroup>

</CodeGroup>

## Connection signals

<CodeGroup>
```gdscript Godot
func _ready():
    WavedashSDK.backend_connected.connect(_on_connected)
    WavedashSDK.backend_disconnected.connect(_on_disconnected)
    WavedashSDK.backend_reconnecting.connect(_on_reconnecting)

func _on_connected(payload):
    hide_connection_indicators()

func _on_disconnected(payload):
    show_offline_indicator()

func _on_reconnecting(payload):
    show_reconnecting_indicator()
```

```csharp Unity
// Backend connection events are not yet available in the Unity SDK.
// Use SDK.IsReady() to check connection status.
```

```javascript JavaScript
// Connection signals:
// BACKEND_CONNECTED - Connected to Wavedash backend
// BACKEND_DISCONNECTED - Lost connection
// BACKEND_RECONNECTING - Attempting to reconnect
```
</CodeGroup>

| Signal | Description |
|--------|-------------|
| `BACKEND_CONNECTED` | Connected to Wavedash backend |
| `BACKEND_DISCONNECTED` | Lost connection |
| `BACKEND_RECONNECTING` | Attempting to reconnect |
